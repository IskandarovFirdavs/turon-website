{% trans "Turon Tarixi" %}

{% trans "Turon (Turon jang san’ati, Yakkakurash Turon, 'Turon yakkakurashi')" %}

{% trans "Turon — O‘zbekistonning milliy sport turi bo‘lib, qadimiy turkiy xalqlarning jang san’atlariga asoslangan. Ushbu sport turi zamonaviy qoidalar asosida 1991-yilda Bahodir Saidov tomonidan shakllantirilgan va 1992-yilda O‘zbekiston sport turlari ro‘yxatiga rasmiy ravishda kiritilgan." %}

{% trans "Turon sport turi faqat Xalqaro Turon Federatsiyasi tomonidan boshqariladi va musobaqalarni o‘tkazish hamda ularda ishtirok etish shartlari, talab va tartiblarini belgilovchi Qoidalar bilan tartibga solinadi." %}

{% trans "Sport turining nomi mazmuni" %}

{% trans "'Turon' so‘zining (ildizi 'Tur', 'Tura') qadimgi tillarda (turk, fors, sanskrit) ma’nosi chaqqonlik, kuch, jasorat kabi fazilatlar bilan ajralib turuvchi qahramon tushunchasini bildirgan. Shu bois, 'Turon' nomining mazmuni sportning mohiyatiga mos keladi — ya’ni yuqori natijalarga erishish uchun sportchi yetarli darajada chaqqonlik, kuch hamda axloqiy va irodaviy sifatlarga ega bo‘lishi lozim." %}

{% trans "Turonning umumiy xususiyatlari" %}

{% trans "Turon tibbiy va sog‘lomlashtiruvchi gimnastika, sport mashg‘ulotlari hamda ma’naviy-axloqiy qadriyatlar rivojini o‘zida mujassam etadi:" %}

  • {% trans "Turon bilan shug‘ullanish o‘g‘il va qiz bolalarga 5–6 yoshdan boshlab ruxsat etiladi." %}
  • {% trans "Turonning asosiy maqsadi — sog‘lomlashtirish va jismoniy tarbiya orqali tanani chiniqtirishdir." %}
  • {% trans "Aql-zakovat, axloqiy va irodaviy fazilatlarni rivojlantirish." %}
  • {% trans "Fikrlash va o‘zini boshqarish qobiliyatini takomillashtirish." %}

{% trans "Turonning falsafiy asoslari" %}

{% trans "Turon bilan shug‘ullanuvchilar uchun xulq madaniyatini o‘rganish katta ahamiyat kasb etadi va tarbiyaviy jarayonda muhim o‘rin tutadi. Insoniylik, adolat, ezgulik kabi yuksak axloqiy tamoyillarga asoslangan xulq madaniyati Turon bilan shug‘ullanuvchilarda halollik, sezgirlik, boshqalarga hurmat va yuqori darajadagi o‘zini tuta bilish sifatlarini shakllantiradi." %}
